# bilibili（B站移动端开发） --  rem + flex

## 技术栈  

* vue + vue-cli + vue-router + axios + ES6 + sass + vant + flex + rem  

## 项目运行  

```
cd bilibili
```

```
npm install
```

```
npm run serve
```
